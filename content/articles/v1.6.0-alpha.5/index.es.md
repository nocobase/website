### 🐛 Corrección de errores

- **[cliente]**
  - Se corrige el problema por el cual el título de la pestaña del navegador no se actualiza al cambiar de página ([#5857](https://github.com/nocobase/nocobase/pull/5857)) por @zhangzhonghe

  - Se corrige el problema por el cual al actualizar la página en la página de gestión de fuentes de datos se redirige a la página de inicio ([#5855](https://github.com/nocobase/nocobase/pull/5855)) por @zhangzhonghe

  - Se corrige el problema de que los datos no cambian después de la paginación en una subtabla ([#5856](https://github.com/nocobase/nocobase/pull/5856)) por @zhangzhonghe

  - Se corrige el problema por el cual los datos de roles no se muestran en la tabla de gestión de usuarios ([#5846](https://github.com/nocobase/nocobase/pull/5846)) por @zhangzhonghe

  - Se corrige el problema por el cual los campos de asociación en plantillas de referencia a veces no muestran datos ([#5848](https://github.com/nocobase/nocobase/pull/5848)) por @zhangzhonghe

  - Se corrige el problema por el cual el botón 'Solicitud personalizada' no es visible inmediatamente después de agregarse ([#5845](https://github.com/nocobase/nocobase/pull/5845)) por @zhangzhonghe

  - Se evita que las páginas ocultas afecten las interacciones con otras páginas ([#5836](https://github.com/nocobase/nocobase/pull/5836)) por @zhangzhonghe

  - Se corrige el problema por el cual los bloques agregados en una ventana emergente no se muestran al volver a abrir la ventana ([#5838](https://github.com/nocobase/nocobase/pull/5838)) por @zhangzhonghe

- **[servidor]** Se establecen las acciones disponibles predeterminadas para el ACL ([#5847](https://github.com/nocobase/nocobase/pull/5847)) por @chenos
