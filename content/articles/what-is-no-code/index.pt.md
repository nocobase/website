## **O que é Low-Code?**

Desenvolvimento low-code é uma abordagem simplificada para a criação de software. Este método permite que equipes criem soluções digitais e aplicativos empresariais com o mínimo de codificação.

Plataformas low-code fornecem aos usuários um kit de ferramentas. Este kit inclui componentes pré-construídos e scripts de modelo. Os usuários podem desenvolver novos processos e aplicativos rapidamente. Não há necessidade de escrever código extenso ou testar scripts.

Essas plataformas possuem interfaces intuitivas de arrastar e soltar. Este design permite que qualquer pessoa em uma organização desenvolva aplicativos de negócios. Nenhuma experiência especializada em codificação é necessária.

## **Low-Code é um Gerador de Código?**

Embora as plataformas low-code possam funcionar como geradores de código, elas abrangem muito mais do que os geradores de código tradicionais. As plataformas low-code não apenas geram código; elas oferecem um ambiente de desenvolvimento e um kit de ferramentas abrangentes. Isso permite que os usuários criem, testem e implantem aplicativos rapidamente por meio de uma interface gráfica, aumentando a eficiência e a acessibilidade para desenvolvedores e não desenvolvedores.

## **Quem Deve Usar Low-Code?**

As plataformas low-code são ideais para profissionais de TI e equipes com formação técnica. Essas plataformas permitem o desenvolvimento rápido de aplicativos. Elas fornecem acesso e controle sobre o código. Isso é ideal para desenvolvimentos mais complexos ou personalizados.

Essas plataformas são ferramentas poderosas para desenvolvedores. Elas aceleram o ciclo de desenvolvimento. Os desenvolvedores ainda podem ajustar e personalizar seus aplicativos conforme necessário.

## **O que é No-Code?**

No-code é uma abordagem de desenvolvimento de software projetada para eliminar ou reduzir drasticamente a necessidade de codificação tradicional. Ela permite que usuários não técnicos criem e configurem aplicativos usando ferramentas de programação visual e interfaces de arrastar e soltar.

As plataformas no-code capacitam os usuários a criar aplicativos complexos rapidamente. Os usuários não precisam se aprofundar em linguagens de programação ou técnicas de desenvolvimento. Essa acessibilidade acelera a transformação digital em vários setores, tornando o desenvolvimento de aplicativos mais inclusivo.

## **Qual é a Diferença Entre No-Code e Low-Code?**

### **Plataformas No-Code**

* **Casos de Uso:** Geralmente adequadas para criar aplicativos relativamente simples, como formulários de coleta de dados e pequenos sistemas de gestão. Essas plataformas podem ter limitações quando se trata de sistemas mais complexos ou aplicativos altamente personalizados.
* **Filosofia de Design:** Visam minimizar ou eliminar completamente a necessidade de escrever código. Os usuários constroem aplicativos arrastando e soltando componentes e definindo configurações, sem necessidade de interagir com código.
* **Ambiente de Desenvolvimento:** Oferecem um ambiente de desenvolvimento altamente padronizado, com menor liberdade no design e na funcionalidade.

### **Plataformas Low-Code**

* **Casos de Uso:** Podem suportar o desenvolvimento de aplicativos mais complexos, incluindo aplicativos de nível empresarial e sistemas que exigem processamento de dados complexo ou lógica de negócios. Os desenvolvedores podem construir rapidamente as partes fundamentais usando as ferramentas da plataforma, enquanto abordam problemas mais complexos por meio de codificação.
* **Filosofia de Design:** Embora também forneçam ferramentas de desenvolvimento visual, os usuários podem precisar escrever algum código em certas situações para alcançar funcionalidades ou integrações específicas. Elas oferecem editores de código e opções de personalização mais avançadas.
* **Ambiente de Desenvolvimento:** Simplifica o processo de desenvolvimento, mas permite mais personalização e extensão. Os desenvolvedores podem se aprofundar nas configurações do sistema e realizar personalizações no nível do código.

## **Quem Deve Usar No-Code?**

As plataformas no-code são ideais para usuários não técnicos. Elas são adequadas para analistas de negócios, profissionais de marketing e profissionais de vários setores.

Essas plataformas oferecem uma interface gráfica completamente visual, permitindo que usuários sem qualquer conhecimento de programação criem e implantem aplicativos.

## **O que é NocoBase?**

A NocoBase é uma plataforma no-code com inteligência artificial, projetada para ser fácil e extensivamente extensível.

Ela combina a simplicidade e facilidade de uso do no-code com a ampla flexibilidade do desenvolvimento nativo. O princípio de design da NocoBase é: 80% das necessidades podem ser atendidas por meio de soluções no-code, enquanto os 20% restantes podem ser alcançados por meio de desenvolvimento estendido. É por isso que a NocoBase adota o conceito de "[Tudo é implementado como plugins.](https://www.nocobase.com/en/plugins)"

Os plugins são plug-and-play, permitindo que os usuários desenvolvam novos plugins para atender a novos requisitos, seja para páginas, blocos, operações, APIs ou fontes de dados.

## **A NocoBase é uma Plataforma Low-Code?**

Na NocoBase, você pode escrever lógica de frontend usando blocos JS. No entanto, a NocoBase não é um gerador de código e não suporta exportação de código. Para estender sua funcionalidade, você pode desenvolver plugins personalizados.

## **A NocoBase é uma Plataforma No-Code?**

Sim, a NocoBase é uma plataforma no-code. Ela permite o desenvolvimento de sistemas de negócios complexos e exclusivos, mas continua sendo muito simples de usar.

Os usuários podem alternar entre o modo de uso e o modo de configuração com um clique. A página atua como uma tela onde vários blocos e operações podem ser colocados e combinados para criar a interface desejada, semelhante ao Notion. O modo de configuração de interface é projetado para usuários comuns, não para programadores.

## **A NocoBase é a Mesma Coisa que NocoDB?**

Não, esses são produtos completamente diferentes. [NocoBase](https://docs.nocobase.com/welcome/introduction) é uma plataforma de desenvolvimento no-code orientada por modelo de dados, enquanto NocoDB é um banco de dados no-code orientado por formulários.
