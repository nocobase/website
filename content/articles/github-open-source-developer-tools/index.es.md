## I. Herramientas de Desarrollo Low-Code/No-Code

> **Propósito principal**: Crear rápidamente sistemas back-end, sistemas empresariales, herramientas internas, paneles de gestión y aplicaciones basadas en datos, reduciendo la barrera de desarrollo mediante la visualización o configuración.
>
> **Público objetivo**: Desarrolladores, personal de negocio, ingenieros full-stack

### 1.**Streamlit (⭐41.1k)**

![Streamlit.PNG](https://static-docs.nocobase.com/bd019a12f9407f8a324ffca8f4eccad0.PNG)

**GitHub**: https://github.com/streamlit/streamlit

**Sitio web**: https://streamlit.io/

**Introducción**: Un framework web de Python centrado en la visualización de datos que permite a los desarrolladores transformar rápidamente la lógica de procesamiento de datos en aplicaciones web interactivas.

**Características principales**:

* Desarrollo de scripts minimalista: Utilizando una API minimalista, los desarrolladores pueden crear una aplicación web con solo unas pocas líneas de código Python. La aplicación se actualiza y refresca automáticamente cada vez que se guarda el código, lo que la hace adecuada para la iteración rápida y la experimentación.
* Integración de interacción nativa: Agregar componentes interactivos es tan simple como definir variables. No es necesario escribir código back-end, configurar rutas, manejar solicitudes ni lidiar con HTML/CSS/JS, lo que reduce la barrera para el desarrollo full-stack.
* Implementación inmediata: Las aplicaciones se pueden implementar directamente en Streamlit Community Cloud (gratuito), Snowflake (nivel empresarial) u otras plataformas para un lanzamiento y uso compartido rápidos.

* Integración de múltiples fuentes de datos: Soporta de forma nativa más de 25 tipos de bases de datos y cualquier API REST o GraphQL, conectándose sin problemas a los sistemas empresariales existentes.
