## I. Ferramentas de Desenvolvimento Low-Code/No-Code

> **Propósito Principal**: Construir rapidamente sistemas de back-end, sistemas de negócios, ferramentas internas, painéis de gerenciamento e aplicações orientadas a dados, reduzindo o limite de desenvolvimento através de visualização ou configuração.
>
> **Público-Alvo**: Desenvolvedores, pessoal de negócios, engenheiros full-stack

### 1.**Streamlit (⭐41,1k)**

![Streamlit.PNG](https://static-docs.nocobase.com/bd019a12f9407f8a324ffca8f4eccad0.PNG)

**GitHub**: https://github.com/streamlit/streamlit

**Website**: https://streamlit.io/

**Introdução**: Um framework web Python focado em visualização de dados que permite aos desenvolvedores transformar rapidamente a lógica de processamento de dados em aplicações web interativas.

**Principais Características**:

* Desenvolvimento de Scripts Minimalista: Usando uma API minimalista, os desenvolvedores podem construir uma aplicação web com apenas algumas linhas de código Python. A aplicação é atualizada e recarregada automaticamente sempre que o código é salvo, tornando-a adequada para iteração e experimentação rápidas.
* Integração de Interação Nativa: Adicionar componentes interativos é tão simples quanto definir variáveis. Não há necessidade de escrever código de back-end, configurar rotas, lidar com requisições ou lidar com HTML/CSS/JS, reduzindo o limite para o desenvolvimento full-stack.
* Implantação Imediata: As aplicações podem ser implantadas diretamente no Streamlit Community Cloud (gratuito), Snowflake (nível empresarial) ou outras plataformas para lançamento e compartilhamento rápidos.
