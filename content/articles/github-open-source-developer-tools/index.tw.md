![Zeal.png](https://static-docs.nocobase.com/Zeal-wliub3.png)

**GitHub**：https://github.com/zealdocs/zeal

**網站**：https://zealdocs.org/

**介紹**：一個文件管理器（類似 Dash 的風格），支援快速搜尋本地技術文件，方便開發者查閱資訊。

**核心功能**

* 豐富的 Docsets 支援：附帶 180-200 個內建和可下載的 docsets，滿足大多數開發需求。
* 快速離線搜尋與編輯器整合：支援離線搜尋，無需網路即可快速存取文件；支援快捷鍵觸發查詢介面，並可與 Emacs、Vim、Sublime 等編輯器整合。
* 靈活的搜尋與高擴展性：允許同時跨多個 docsets 搜尋，並支援基於語法的過濾；讓使用者可以自訂或導入 Dash 格式的新 Docsets。
