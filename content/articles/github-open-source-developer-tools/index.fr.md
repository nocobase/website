## I. Outils de développement Low-Code/No-Code

> **Objectif principal** : Créer rapidement des systèmes back-end, des systèmes métier, des outils internes, des panneaux de gestion et des applications axées sur les données, réduisant le seuil de développement grâce à la visualisation ou à la configuration.
>
> **Public cible** : Développeurs, personnel métier, ingénieurs full-stack

### 1.**Streamlit (⭐41,1k)**

![Streamlit.PNG](https://static-docs.nocobase.com/bd019a12f9407f8a324ffca8f4eccad0.PNG)

**GitHub** : https://github.com/streamlit/streamlit

**Site Web** : https://streamlit.io/

**Présentation** : Un framework web Python axé sur la visualisation de données qui permet aux développeurs de transformer rapidement la logique de traitement des données en applications web interactives.

**Fonctionnalités principales** :

* Développement de scripts minimaliste : Grâce à une API minimaliste, les développeurs peuvent créer une application web avec seulement quelques lignes de code Python. L'application se rafraîchit et se met à jour automatiquement à chaque sauvegarde du code, ce qui la rend idéale pour l'itération rapide et l'expérimentation.
* Intégration native des interactions : L'ajout de composants interactifs est aussi simple que la définition de variables. Il n'est pas nécessaire d'écrire du code back-end, de configurer des routes, de gérer les requêtes ou de s'occuper du HTML/CSS/JS, ce qui réduit le seuil du développement full-stack.
* Déploiement immédiat : Les applications peuvent être directement déployées sur Streamlit Community Cloud (gratuit), Snowflake (niveau entreprise) ou d'autres plateformes pour un lancement et un partage rapides.

**Fonctionnalités principales** :

* Intégration multi-sources de données : Elle prend nativement en charge plus de 25 types de bases de données et toute API REST ou GraphQL, se connectant de manière transparente aux systèmes d'entreprise existants.
* Capacités d'IA et d'automatisation : Elle dispose de fonctions d'IA intégrées et d'un moteur de workflow, prenant en charge la construction d'agents intelligents (Agents) et de processus métier automatisés, accélérant le développement d'applications intelligentes.

* Capacité d'intégration au niveau entreprise : Il prend en charge les services back-end grand public tels que GraphQL et REST, adapté à la construction de systèmes de gestion B2B ou SaaS complexes.
