## I. Công cụ Phát triển Low-Code/No-Code

> **Mục đích cốt lõi**: Nhanh chóng xây dựng hệ thống back-end, hệ thống kinh doanh, công cụ nội bộ, bảng quản lý và ứng dụng dựa trên dữ liệu, giảm ngưỡng phát triển thông qua trực quan hóa hoặc cấu hình.
>
> **Đối tượng mục tiêu**: Nhà phát triển, nhân viên kinh doanh, kỹ sư full-stack

### 1.**Streamlit (⭐41.1k)**

![Streamlit.PNG](https://static-docs.nocobase.com/bd019a12f9407f8a324ffca8f4eccad0.PNG)

**GitHub**: https://github.com/streamlit/streamlit

**Website**: https://streamlit.io/

**Giới thiệu**: Một framework web Python tập trung vào trực quan hóa dữ liệu cho phép các nhà phát triển nhanh chóng biến logic xử lý dữ liệu thành các ứng dụng web tương tác.

**Tính năng cốt lõi**:

* Phát triển Script Tối giản: Sử dụng API tối giản, các nhà phát triển có thể xây dựng một ứng dụng web chỉ với vài dòng mã Python. Ứng dụng tự động làm mới và cập nhật mỗi khi mã được lưu, phù hợp cho việc lặp lại và thử nghiệm nhanh chóng.
* Tích hợp Tương tác Gốc: Việc thêm các thành phần tương tác đơn giản như định nghĩa biến. Không cần viết mã back-end, thiết lập route, xử lý request hay đụng đến HTML/CSS/JS, giảm ngưỡng cho phát triển full-stack.
* Triển khai Ngay lập tức: Các ứng dụng có thể được triển khai trực tiếp lên Streamlit Community Cloud (miễn phí), Snowflake (cấp doanh nghiệp) hoặc các nền tảng khác để ra mắt và chia sẻ nhanh chóng.

**Tính năng cốt lõi**:

* Khả năng AI và Tự động hóa: Có sẵn các chức năng AI và công cụ quy trình làm việc, hỗ trợ xây dựng các tác nhân thông minh (Agents) và quy trình kinh doanh tự động, tăng tốc phát triển ứng dụng thông minh.
