Basándonos en los comentarios de los usuarios, hemos decidido convertir en código abierto dos plugins comerciales de flujo de trabajo:

* [Cálculo JSON](https://v2.docs.nocobase.com/workflow/nodes/json-query)
* [Mapeo de variables JSON](https://v2.docs.nocobase.com/workflow/nodes/json-variable-mapping)

## ¿Por qué código abierto?

Estos plugins se encuentran entre los más utilizados en los flujos de trabajo y a menudo son necesarios después de los nodos SQL o de Solicitud HTTP. Cuando eran plugins comerciales, los usuarios de la edición comunitaria no podían utilizar completamente el nodo de Solicitud HTTP, lo que generaba una mala experiencia. Por eso los estamos convirtiendo en código abierto.

## Resumen del plugin

### Cálculo JSON

Maneja JSON complejo producido por nodos ascendentes, permitiéndole calcular, extraer o remodelar los datos.

Los casos de uso comunes incluyen:

* Resultados de consultas SQL
* Datos devueltos por solicitudes HTTP

Ayuda a convertir estas salidas en los formatos de variable y valores de campo necesarios para los nodos descendentes.

### Mapeo de variables JSON

Asigna estructuras JSON complejas a variables que son más fáciles de usar en pasos posteriores del flujo de trabajo.

Los casos de uso comunes incluyen:

* Extraer campos clave de JSON anidado devuelto por API o SQL
* Renombrar campos o ajustar estructuras de arrays
* Preparar variables más limpias para nodos subsiguientes

Con este plugin, evita tener que configurar repetidamente rutas de campo profundas en cada nodo.

## Ajustes para clientes existentes

Sus derechos permanecerán sin cambios. Reembolsaremos puntos de plugin de la siguiente manera:

* *Uso vitalicio, actualización de 1 año* → 2 puntos
* *Uso vitalicio y actualización* → 4 puntos

Estos ajustes se completarán para todos los clientes existentes antes del 23 de noviembre de 2025, y la empresa matriz de NocoBase proporcionará la interpretación final.

## Para concluir

Gracias a todos los usuarios de NocoBase y contribuyentes de código abierto. Sus comentarios y problemas nos ayudan a mejorar con cada versión. Continuaremos avanzando más rápido y ofreciendo capacidades más sólidas a un precio más amigable.
