根據用戶反饋，我們決定將兩個商業工作流插件開源：

* [JSON 計算](https://v2.docs.nocobase.com/workflow/nodes/json-query)
* [JSON 變量映射](https://v2.docs.nocobase.com/workflow/nodes/json-variable-mapping)

## 為何開源？

這些插件是工作流中最常用的插件之一，且通常在 SQL 或 HTTP 請求節點之後使用。當它們還是商業插件時，社區版用戶無法完整使用 HTTP 請求節點，導致體驗不佳。因此我們決定將其開源。

## 插件概述

### JSON 計算

處理上游節點產生的複雜 JSON，讓您能夠計算、提取或重塑數據。

常見使用場景包括：

* SQL 查詢結果
* HTTP 請求返回的數據  
它能幫助將這些輸出轉換為下游節點所需的變量格式與欄位值。

### JSON 變量映射

將複雜的 JSON 結構映射為變量，方便在後續工作流步驟中使用。

常見使用場景包括：

* 從 API 或 SQL 返回的巢狀 JSON 中提取關鍵欄位
* 重新命名欄位或調整陣列結構
* 為後續節點準備更乾淨的變量

使用此插件，您無需在每個節點中重複配置深層欄位路徑。

## 對現有客戶的調整

您的權益將保持不變。我們將按以下方式退還插件點數：

* *終身使用，1 年升級* → 2 點
* *終身使用與升級* → 4 點

這些調整將於 2025 年 11 月 23 日前完成所有現有客戶的處理，最終解釋權歸 NocoBase 母公司所有。
