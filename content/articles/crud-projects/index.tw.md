**主要功能**

* **自動化 CRUD 操作**：APIJSON 提供強大的 CRUD 功能，支援透過簡單的 JSON 配置進行複雜查詢，包括關聯查詢和複雜的過濾條件。
* **安全性和靈活性**：內建自動權限驗證和 SQL 注入保護機制，確保 API 安全。APIJSON 也支援版本管理，因此 API 迭代不會影響現有系統。
* **高效的資料處理**：支援在單次請求中獲取多種資源和資料形式，減少了多次請求的需求，並優化了資料互動效率。

**使用案例**

* **快速開發**：適用於需要快速建立原型和開發應用程式的新創公司和中小型企業，特別是在專案需要快速迭代和頻繁更改 API 時。
* **內部工具**：為企業管理系統和監控系統快速建立後端管理介面。
* **教育與學習**：由於其易用性，APIJSON 也適用於教育環境，幫助學生和初學者理解 API 的建構和使用。

PHP-CRUD-API 是一個單一檔案的 PHP 腳本，透過簡單的配置即可為 SQL 資料庫添加完整的 REST API。它支援多種資料庫系統，包括 MySQL/MariaDB、PostgreSQL、SQL Server 和 SQLite，使其成為快速部署 API 的理想選擇。

PHP-CRUD-API 提供了一種高效的方式來建立和管理資料庫驅動的 REST API，無需編寫大量的後端程式碼，使其成為快速開發 Web 應用程式的強大工具。

**主要功能**

* **單一檔案部署**：整個 API 透過一個 PHP 檔案運行，無需複雜的安裝或配置。
* **多資料庫支援**：與多種資料庫相容，透過簡單配置即可連接到不同類型的資料庫。
* **豐富的功能**：支援 CRUD 操作、自動文件生成（OpenAPI）、複雜查詢和資料權限管理。
* **可擴展性**：支援中介軟體、自訂控制器和環境變數配置，允許根據需要調整和擴展功能。

**使用案例**

* **快速原型開發**：對於需要快速驗證資料驅動應用程式的開發者，可以在短時間內部署和測試 API。
* **小型專案**：適用於需要快速且直接後端解決方案的中小型專案。
