**Visão Geral**

NocoBase é uma plataforma de desenvolvimento no-code orientada a modelo de dados, projetada para construir ferramentas internas. Sua arquitetura baseada em plugins é altamente extensível e suporta a construção de painéis administrativos, sistemas de negócios, CRMs, fluxos de aprovação e uma ampla gama de aplicações internas. Oferece um construtor de interface visual WYSIWYG e suporta relacionamentos de dados complexos, modelos de permissão e lógica personalizada, sendo ideal para equipes que precisam entregar ferramentas de negócios rapidamente.

No NocoBase, a IA não é um módulo isolado. Em vez disso, ela é integrada diretamente em tabelas, formulários, modelos de dados e fluxos de trabalho. O framework AI Employees integra IA às operações de negócios, permitindo a colaboração com modelos, formulários, tabelas e fluxos de trabalho. Os tipos de funcionários de IA integrados incluem limpeza de dados, modelagem de esquemas, pesquisa de conhecimento, geração de gráficos e geração de código, ajudando os usuários a reduzir o custo do processamento de dados, configuração de interface e construção de lógica.

**Casos de Uso**

**Limpeza de dados (Dex, Especialista em Processamento de Dados)**

Funcionários de IA extraem informações estruturadas de conteúdo não estruturado, como dados de clientes, tickets ou texto de contratos, e podem preencher formulários automaticamente.

![Dex.gif](https://static-docs.nocobase.com/Dex-c4uwn4.gif)

**Análise de dados (Viz, Analista de Insights)**

Em páginas como Leads, Oportunidades ou Contas, os funcionários de IA podem analisar o conjunto de dados atual e criar automaticamente gráficos de tendências, comparações, cartões de KPI e insights concisos.

![Viz.gif](https://static-docs.nocobase.com/Viz-demgin.gif)

**Configuração de frontend (Nathan, Engenheiro Frontend)**

Ao trabalhar com JSBlock, JSField, JSItem, JSColumn, JSAction ou Fluxo de Eventos, os funcionários de IA podem revisar o contexto do código e gerar ou refinar JavaScript com base em prompts de linguagem natural.

![Nathan.png](https://static-docs.nocobase.com/Nathan-n9ybjn.png)

**Modelagem de dados (Orin, Modelador de Dados)**

Ao criar sistemas internos como CRMs, tickets, inventário ou gerenciamento de projetos, os funcionários de IA podem ajudar a gerar ou atualizar tabelas de dados.

**Visão Geral**

Budibase é uma plataforma low code open source para construir ferramentas internas. Ela suporta conexões de banco de dados, múltiplas fontes de dados, criação de UI por arrastar e soltar e geração automática de CRUD e fluxos de trabalho, sendo adequada para construir backends empresariais, fluxos de aprovação, sistemas de formulários e painéis administrativos. Com seu banco de dados integrado, suporte a API REST e construtor visual, equipes sem ampla experiência em frontend ainda podem criar aplicações internas completas.

O Budibase também integrou a IA como uma capacidade nativa da plataforma. Com o Budibase AI habilitado, os usuários podem criar Colunas de IA para operações inteligentes de dados, gerar lógica JavaScript a partir de linguagem natural, produzir automaticamente esquemas de banco de dados e usar automações baseadas em IA para limpeza, classificação, transformação e geração de dados.

**Casos de Uso**

**Limpeza inteligente de dados e preparação de conteúdo**

As Colunas de IA podem limpar, normalizar e categorizar dados automaticamente, úteis para feedback de clientes, moderação de conteúdo, processamento de tickets e fluxos de trabalho semelhantes.

**Geração automática de esquemas de banco de dados**

Ao construir novos módulos, como gerenciamento de clientes, fluxos de trabalho de contratos ou sistemas de tarefas, o Budibase AI pode gerar estruturas de tabela adequadas e registros de exemplo a partir de um único prompt de linguagem natural.

**Visão Geral**

Refine é um meta-framework React open source construído para aplicações web com uso intensivo de CRUD, sistemas administrativos empresariais, painéis e outras ferramentas internas B2B.

O Refine AI traz IA para o fluxo de trabalho de desenvolvimento. Com linguagem natural, documentação ou especificações de API como entrada, o Refine AI pode gerar um scaffold completo de aplicação React pronta para produção, incluindo páginas CRUD, roteamento, autenticação, gerenciamento de estado e busca de dados. A saída é um código React limpo e padrão que os desenvolvedores podem estender ou adaptar livremente.

**Casos de Uso**

**Geração de sistemas administrativos**

Com base em APIs ou modelos de dados, o Refine AI pode criar interfaces CRUD com páginas de lista, formulário e detalhes, sendo ideal para inicializar rapidamente sistemas como inventário, pedidos ou gerenciamento de projetos.

**Refatoração e manutenção de bases de código existentes**

Como o Refine AI gera código que segue padrões de nível empresarial e melhores práticas, ele também pode ser usado para refatorar sistemas mais antigos em componentes modulares ou para adicionar novos módulos de negócios rapidamente em cima de um projeto existente, melhorando tanto a velocidade de desenvolvimento quanto a manutenibilidade a longo prazo.

Ela oferece um construtor visual de arrastar e soltar, um banco de dados integrado, extensas integrações de fontes de dados e SaaS, e recursos integrados de permissão e colaboração. O ToolJet permite que as equipes criem ferramentas internas descrevendo requisitos em linguagem natural. Ele pode construir UI, estruturas de dados e lógica do zero ou modificar aplicações existentes. Isso facilita a produção de sistemas internos comuns, como gerenciamento de inventário, em um curto espaço de tempo.

**Visão Geral**

Appsmith é uma plataforma low code open source construída para ajudar equipes a criar aplicações internas rapidamente e com o mínimo de código. Ela suporta uma ampla gama de bancos de dados e APIs, fornece uma biblioteca abrangente de componentes de UI visuais e permite que os desenvolvedores estendam a funcionalidade com JavaScript. Isso a torna adequada para construir sistemas administrativos, painéis de dados, fluxos de trabalho de aprovação, CRMs e ferramentas de análise.

A versão mais recente introduz suporte nativo a IA. O Appsmith pode se conectar a LLMs líderes como OpenAI, Google AI e Anthropic, permitindo recursos como análise de texto, sumarização, pesquisa semântica e classificação alimentados por ações de IA.

**Casos de Uso**

**Acelerando o desenvolvimento com IA**

O assistente de IA do Appsmith fornece suporte contextual durante o desenvolvimento, oferecendo orientação sobre operações, recursos da plataforma, construção de consultas, lógica JavaScript e personalização de UI. Através do assistente lateral Appy, os desenvolvedores podem obter ajuda relevante diretamente na página que estão editando, reduzindo o tempo de consulta à documentação e acelerando a iteração.

![借助 AI 提升构建速度.png](https://static-docs.nocobase.com/%E5%80%9F%E5%8A%A9%20AI%20%E6%8F%90%E5%8D%87%E6%9E%84%E5%BB%BA%E9%80%9F%E5%BA%A6-76dac4.png)

**Incorporando IA em fluxos de trabalho internos**

As equipes podem construir ferramentas internas aumentadas por IA, como um painel de análise de avaliações de clientes. O Appsmith pode buscar dados de avaliação, exibi-los em tabelas, executar análise de sentimentos por meio da fonte de dados de IA do Appsmith e usar Objetos JS para escrever os resultados de volta na UI. Isso permite que as equipes classifiquem automaticamente o feedback e priorizem problemas com mais eficiência.

![整合 AI 能力.png](https://static-docs.nocobase.com/%E6%95%B4%E5%90%88%20AI%20%E8%83%BD%E5%8A%9B-n2t9w0.png)

Além disso, além dos cinco grandes projetos de alto star acima, ferramentas open source como [Yao](https://github.com/YaoApp/yao) e [ILLA Builder](https://github.com/illacloud/illa-builder) também mantêm forte atividade na comunidade. [Yao](https://github.com/YaoApp/yao) fornece um mecanismo de aplicação full-stack que permite a construção rápida de modelos, fluxos de trabalho e APIs para sistemas internos. Ambos os projetos também oferecem extensões básicas de IA e podem ser opções complementares úteis quando as equipes estão construindo ferramentas internas.

### Conclusão

Ao selecionar ferramentas internas, é útil avaliar as soluções em várias dimensões.

1. A maturidade dos recursos de IA, como suporte para geração de páginas, geração de SQL ou lógica de negócios, geração de gráficos e assistentes inteligentes.
2. Atividade da comunidade e do ecossistema, incluindo Stars, Issues, Commits e atualizações de documentação, que sinalizam a confiabilidade e sustentabilidade a longo prazo de um projeto.
3. Capacidades de auto-hospedagem e segurança, que influenciam a implantação interna e a conformidade de dados.
4. Extensibilidade e facilidade de personalização, que afetam diretamente o custo contínuo de adaptação dos sistemas às necessidades de negócios em evolução.

Esperamos que este guia ofereça uma visão técnica útil e ajude você a tomar decisões mais informadas ao construir ferramentas internas.

À medida que a IA generativa continua a avançar, esses projetos open source continuarão a evoluir também, trazendo novas capacidades ao longo do tempo.
