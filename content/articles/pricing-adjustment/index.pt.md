---
title: "Ajuste de preços do NocoBase"
description: "Ajustamos os preços para plugins comerciais, pacotes de plugins comerciais e benefícios das edições comerciais. Além disso, oferecemos ajustes gratuitos para usuários que já realizaram compras."
---

NocoBase foi lançado no GitHub em março de 2021 e, após três anos de desenvolvimento comunitário, a versão 1.0 foi lançada em 28 de abril de 2024, oferecendo oficialmente edições e serviços comerciais. Nos 11 meses desde a versão 1.0, lançamos um total de 229 versões nos branches Main, Beta e Alpha, garantindo licenças comerciais de quase 200 empresas em 32 países ao redor do mundo. Como um produto de código aberto, esses clientes comerciais nos forneceram a base para a sobrevivência e são cruciais para o nosso desenvolvimento contínuo. Com o apoio desses usuários, alcançamos um estado de lucratividade estável.

Planejamos lançar o NocoBase 2.0 em breve, que incluirá recursos significativos, como funcionários de IA. Para agradecer a esses quase 200 clientes comerciais e permitir que mais usuários acessem os recursos avançados do NocoBase a um custo menor, decidimos reduzir o preço dos plugins comerciais e aumentar os benefícios das edições comerciais.

### Os ajustes nos [plugins comerciais](https://www.nocobase.com/en/plugins-commercial) são os seguintes:

* Alguns plugins serão alterados de comerciais para gratuitos. Por exemplo: Workflow - Response Message Node, Departments.
* O preço de todos os outros plugins será reduzido para 50% do preço original.

### Os ajustes nos [pacotes de plugins comerciais](https://www.nocobase.com/en/plugins-bundles) são os seguintes:

* Novos pacotes de plugins P8 e P16 serão adicionados com descontos correspondentes, reduzindo a barreira de entrada para pacotes de plugins.
* O preço dos pacotes de plugins existentes P32, P64 e P128 será reduzido para 50% do preço original.
* O preço do pacote de plugins existente P256 será reduzido para 50% do preço original, e os benefícios serão ajustados para um pacote de plugins "Ilimitado", significando acesso a licenças para todos os plugins comerciais atuais e futuros desenvolvidos pela NocoBase.

### Os ajustes nos benefícios das [edições comerciais](https://www.nocobase.com/en/commercial) são os seguintes:

* O número de pontos de plugin incluídos na Edição Standard será aumentado em 100% para 8. As taxas de upgrade subsequentes serão reduzidas para 30% da taxa do primeiro ano.
* O número de pontos de plugin incluídos na Edição Profissional será aumentado em 100% para 64. As taxas de upgrade subsequentes serão reduzidas para 30% (taxa de serviço) + 30% (taxa de upgrade) da taxa do primeiro ano, com a opção de escolher uma ou ambas.
* A Edição Enterprise terá seu pacote de plugins incluído atualizado de P64 para um pacote de plugins "Ilimitado", significando acesso a licenças para todos os plugins comerciais atuais e futuros desenvolvidos pela NocoBase. As taxas de upgrade subsequentes serão reduzidas para 30% (taxa de serviço) + 30% (taxa de upgrade) da taxa do primeiro ano, com a opção de escolher uma ou ambas.

### Os seguintes ajustes gratuitos serão feitos para usuários que já adquiriram licenças de edição comercial, plugins comerciais ou pacotes de plugins comerciais:

* Usuários que adquiriram plugins comerciais (não resgatados usando pontos):
  * Plugins com validade de upgrade de 1 ano: Upgrades vitalícios serão concedidos.
  * Plugins com validade de upgrade vitalícia: Pontos equivalentes a 1 ano de upgrades para o plugin correspondente serão concedidos.
* Usuários que adquiriram pacotes de plugins comerciais:
  * 100% dos pontos do pacote de plugins serão concedidos.
* Usuários que adquiriram licenças de edição comercial (apenas para usuários que não usaram nenhum desconto):
  * Edição Standard: 4 pontos de plugin serão concedidos, e as taxas de upgrade subsequentes serão reduzidas para 30% da taxa do primeiro ano.
  * Edição Profissional: 32 pontos de plugin serão concedidos, e as taxas de upgrade subsequentes serão reduzidas para 30% (taxa de serviço) + 30% (taxa de upgrade) da taxa do primeiro ano, com a opção de escolher uma ou ambas.
  * Edição Enterprise: Um pacote de plugins "Ilimitado" será concedido, e as taxas de upgrade subsequentes serão reduzidas para 30% (taxa de serviço) + 30% (taxa de upgrade) da taxa do primeiro ano, com a opção de escolher uma ou ambas.

Os direitos de interpretação dos ajustes acima pertencem à NocoBase. Os benefícios para usuários que já adquiriram licenças de edição comercial, plugins comerciais ou pacotes de plugins comerciais serão ajustados até 10 de abril.

Mais uma vez, agradecemos a todos os usuários pagantes e gratuitos pelo seu apoio, e agradecemos a todos os contribuidores de código aberto do NocoBase. Continuaremos a iterar rapidamente para fornecer a todos melhores produtos e preços mais baixos.
