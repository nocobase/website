* **Twenty**：以極簡設計和強大的 API 支援為特色，Twenty 非常適合希望快速建立客戶管理系統的中小型企業。

* **NocoBase**：透過資料模型驅動、AI 驅動的無程式碼（零程式碼）平台，企業可以透過基於外掛的架構快速開發客製化的 CRM 系統。
* **Krayin**：一個低程式碼開源 CRM，在易於使用且靈活的框架中提供企業級功能。

* **iDURAR**：一個專注於簡化客戶關係和團隊協作的 CRM 系統，提供完整的客戶管理功能。
* **悟空 CRM**：一個專為中小型企業設計的中文開源 CRM，具備多語言支援和廣泛的整合能力。
* **Dolibarr**：一個全面的開源 ERP 和 CRM 平台，為中小型企業提供完整的業務流程管理。
* **SuiteCRM**：一個功能強大的開源 CRM 系統，具備完整的銷售、行銷和客戶服務自動化功能，提供廣泛的客製化和整合能力。

## 1.[Twenty](https://twenty.com/)

⭐️ 星數：24.4k

GitHub 連結：https://github.com/twentyhq/twenty

![twenty.png](https://static-docs.nocobase.com/a5dfb1229a25a81853c3a39742862015.png)

Twenty 的誕生源於團隊對傳統 CRM 工具的挫折感。作為中小型企業的員工，他們經常被不適合快速成長公司的複雜 CRM 系統所困擾。該團隊決定創建一個「簡單、輕量且高效」的 CRM 解決方案，讓任何企業都能輕鬆管理客戶資料。Twenty 專為中小企業設計，旨在透過精簡的介面和強大的後端支援，使客戶資料輸入和銷售追蹤變得快速且直觀。

**功能特色**：

* **API 支援**：提供廣泛的 API 介面，可與 ERP、電子郵件行銷工具和會計軟體等業務系統整合。
* **資料隱私保護**：客戶資料儲存在公司自己的伺服器上，最大限度地降低第三方資料外洩的風險。
* **行動裝置存取**：採用響應式設計，支援行動裝置的存取和操作。

**理想使用場景**：

Twenty 非常適合預算有限、希望快速建立客戶管理系統的新創公司。它也適用於需要基本 CRM 功能的小型團隊和個人創業者。
