### 🎉 新功能

- **[工作流：審批]** 為審批 API 新增 ACL 控制，作者：@mytharcher
- **[認證：釘釘]** 允許使用 `unionId` 綁定使用者，作者：@2013xile

### 🚀 改進

- **[AI：知識庫]** 優化建置輸出，以減少 plugin-ai-knowledge-base 的套件大小。作者：@cgyrock
- **[多空間]** 多空間權限控制存取 ACL，作者：@jiannx
- **[操作：匯出記錄 Pro]** 根據選取的記錄或資源篩選器，改善匯出操作的資料範圍，作者：@katherinehhh
- **[工作流：審批]**
  - 移除對 JS 欄位的支援，作者：@zhangzhonghe
  - 簡化查詢參數並提升效能，作者：@mytharcher
  - 實作 API 存取控制以防止未經授權的資料操作，作者：@mytharcher
  - 遷移後新增同步受眾的修復邏輯，作者：@mytharcher
- **[認證：釘釘]** 使用 `userid` 作為使用者關聯的預設鍵，同時保持與依賴手機號碼的現有驗證器相容，作者：@2013xile
