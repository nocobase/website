## 每個家都有屬於自己的客製家具故事

問任何一位剛裝修完的人，他們很可能會說：「能省的地方就省，但千萬別在客製櫥櫃上省錢。」

這是因為櫥櫃的安裝方式，以及它們如何有效利用空間，將影響你未來十年甚至更久的日常生活。甲醛超標、衣櫃門板龜裂、櫥櫃縫隙、榻榻米尺寸不合——這些問題都可能成為日後揮之不去的煩惱。

![olmon](https://static-docs.nocobase.com/1-9v80ld.PNG)

客製家具行業出錯是家常便飯，原因很充分——生產、設計和安裝由不同團隊負責，他們依賴傳統溝通方式：電話和聊天群組。環節眾多，幾乎保證會出錯。

全中國有數十萬家小型公司緊跟著客製家具的蓬勃需求。但大多數仍採用紙筆管理，資料混亂、流程雜亂無章是常態。

對消費者來說，「客製」意味著更好的體驗。但對大多數工廠老闆來說，這代表更難的管理、更高的成本和更大的風險。

## 小型企業的隱藏困境

**客製家具行業遠比大多數人想像的複雜。**

對客戶來說，這似乎很簡單——只要告訴設計師你的需求，等上幾週，新家就成形了。

但在幕後，一張訂單可能涉及十幾個步驟：採購原材料、裁切板材、安排生產、協調送貨、現場安裝、售後服務。任何環節都可能出錯——一旦出錯，客戶就會立刻抱怨。

這個挑戰在中國更大，超過 90% 的家具製造商是中小型企業。大多數沒有使用正規的數位系統。相反，他們透過 Excel 試算表、微信群組，甚至手寫筆記來管理一切。設計錯誤、工單遺失、月底帳目不符——這些都是家具工廠每天的頭痛問題。

OLMON 家具只是中國數十萬家小型客製家具公司之一。他們的團隊不大，但每年服務許多客戶——多年來，他們日復一日地面臨所有這些問題。

他們曾考慮購買 ERP 軟體或聘請外部廠商協助。但大多數解決方案不是太貴，就是根本不符合他們的需求。每當討論需求時，答案總是：「這不是標準功能。如果要客製化，需要額外收費。」

「但我們的客戶都有獨特需求！難道每次改系統都要付錢嗎？」對 OLMON 來說，這完全不合理。

自己建系統也行不通——聘請 IT 人員、購買伺服器、持續維護，每年都要花費一大筆錢。

**他們不是看不到數位化的價值——而是對像他們這樣的小企業來說，門檻實在太高了。**

## OLMON 家具如何找到新出路

一切在他們發現 NocoBase 後改變了。

與傳統 ERP 系統不同，NocoBase 是開源的——這讓它價格親民許多——而且採用無程式碼方式，即使非技術人員也能建置和管理系統。「不用寫程式就能建系統」的承諾，給了 OLMON 家具真正的希望。

「這讓他們想：也許值得一試。」

NocoBase 與常見選項不同——它不像傳統程式碼框架那麼昂貴複雜，也不像其他無程式碼或低程式碼平台那麼僵化受限。相反，它結合了兩者的優點：高程式碼的靈活性，加上無程式碼的簡便性。

隨著 OLMON 家具深入研究，他們發現了幾個非常適合他們業務的功能：

* **靈活可控的成本。**

NocoBase 是開源的，免費使用，支援自託管——無需擔心昂貴的訂閱費或隱藏費用。只需為你實際需要的插件付費。

* **超靈活的資料建模。**

所有棘手的業務規則——例如管理客戶、訂單和生產排程之間的多層級關係——都可以透過簡單配置來處理。一對多或多對多等複雜結構（包括組織層級和權限系統）都很容易設定。你甚至可以直接連結 MySQL 或 PostgreSQL 資料庫，無需進行無止盡的資料遷移。

* **無程式碼設定，業務團隊主導。**

業務人員不再需要等待 IT，現在可以透過拖放介面自行修改。工作流程支援多種審批流程，規則變更速度可提升近 90%。數十個內建元件讓設計桌面和行動端介面變得簡單。

* **企業級資料安全。**

透過自託管，你的核心資料與雲端實體隔離，解決了最大的資料隱私問題。細粒度的權限控制——甚至精細到單一欄位，例如讓訂單金額僅對特定角色可見——確保敏感資料受到保護。

* **無限的可擴展性。**

NocoBase 獨特的微核心架構讓系統保持精簡（核心僅 200KB），因此添加新功能（如 BI 分析）只需一個插件。而且憑藉活躍的開源社群，任何客製需求總有解決方案。

## ERP 不再是大公司的專利

OLMON 家具採用 NocoBase 後，逐步將越來越多的營運流程轉移到新的數位平台上。兩個改變遊戲規則的場景脫穎而出——動態業務規則配置和即時生產資料對帳。

### 動態規則配置：告別痛苦的開發週期

使用傳統開發方式，即使業務規則有最微小的變更，技術團隊也要花費數週時間：

* 後端開發人員必須熬夜重寫介面程式碼。
* 前端工程師忙著更新 React 或 Vue 元件。
* 資料庫專家緊張地調整表格結構並遷移舊資料。
* 整個過程可能拖上兩到三週。任何失誤都會導致技術債累積，直到最後沒人願意碰程式碼。

但有了 NocoBase，一切都變了：

* 業務團隊現在可以透過易用的介面直接更新資料模型和業務規則——新的表格和 API 會自動產生。

![新的表格和 API 會自動產生.png](https://static-docs.nocobase.com/2-4bct5b.png)

* 過去需要兩天才能新增一個欄位，現在只需五秒鐘。

![新增一個欄位現在只需五秒鐘](https://static-docs.nocobase.com/3-sufqpf.png)

* 更改一個表單，過去需要三天的來回溝通，現在在平台上十分鐘就能完成。
* 內建的工作流程版本控制讓追蹤變更變得簡單，避免混淆。

![內建的工作流程版本控制](https://static-docs.nocobase.com/4-3yonyu.png)

最重要的是，業務用戶現在掌握主導權，而 IT 只需處理少數核心擴展——他們的工作量大幅減少。

### 手動對帳已成過去式

計件工資在家具行業很常見——但也是薪資糾紛的主要來源。OLMON 家具過去使用紙本單據和 Excel 試算表管理薪資。每個月，財務團隊的五個人要加班整整一週來核對記錄，但錯誤率仍高達 20% 左右，薪資糾紛是家常便飯。

為了解決這個問題，OLMON 將 NocoBase 連接到車間的條碼掃描器和 PDA。現在，每當工人完成一項任務，他們會當場掃描確認——資料直接流入後端系統進行即時驗證：

* 如果工人在十分鐘內掃描同一個條碼，系統會發現重複並阻止。
* MES 和財務資料現在無縫連接，生產進度和薪資始終保持同步。
* 月度對帳從五個人花七天，縮減到一個人花幾小時——錯誤和薪資糾紛幾乎完全消除。

![即時驗證](https://static-docs.nocobase.com/5-067tjj.png)

主管現在可以按小時追蹤即時生產力排名。如果效率波動超過 20%，系統會自動觸發警報，以便快速跟進。

![自動觸發警報](https://static-docs.nocobase.com/6-44482t.png)

透過與金蝶 ERP 等平台的整合，多來源資料即時更新，財務文件自動產生——讓財務管理更加順暢。

至於工人？他們不再需要追著財務問薪資細節——現在他們可以隨時在公司微信上查看自己的每日收入。

## 讓數據說話

隨著數位系統全面到位，OLMON 家具的團隊親眼見證了「效率」不只是口號——而是他們現在每天都能體驗到的事。

那些曾經看似無法解決的問題，現在可以透過清晰可靠的數據進行追蹤、衡量和解決。

| 管理指標          | 之前       | 之後       | 改善幅度   |
| ----------------- | ---------- | ---------- | ---------- |
| 規則調整時間      | 14 天      | 2 小時     | ↓98.5%     |
| 資料錯誤率        | 12%        | 0.30%      | ↓97.5%     |
| 人力成本           | ¥50,000/月 | ¥5,000/月 | ↓90.0%     |
| 客戶投訴率        | 8%         | 0.50%      | ↓93.7%     |

這一切的背後是公司運作方式的徹底轉變。

資料檢查和對帳比以往任何時候都更準確。財務和管理團隊從無止盡的手動工作中解放出來，客戶服務也達到了一個全新的水準。

最重要的是，所有這些改進都來自內部——由 OLMON 自己的員工驅動，而不是昂貴的外部顧問。數位化的力量真正回到了他們自己手中。

## 數位化的真正未來：你負擔得起、真正好用的工具

長期以來，「數位轉型」和「ERP 系統」只屬於大公司。對於像 OLMON 家具這樣無數的傳統小型企業來說，高昂的成本、複雜的開發和漫長的導入過程，讓數位化感覺完全遙不可及。

但現在，一切都不同了。

OLMON 家具並不滿足於「只是數位化」——他們希望將這些新的數位能力轉化為持續創新和成長的基礎。

展望未來，他們的目標很明確：

* **深化技術**

利用 NocoBase 的插件系統建構行業專用功能——例如自動訂單拆分和智慧流程排程——為客製家具優化工作流程。

* **協作生態系統**

與 NocoBase 社群共同為家居客製行業建立共享解決方案庫，推動標準化，讓更多企業更容易分享經驗、互相學習、共同成長——沒有人需要孤軍奮戰。

* **行業願景**

證明即使在像客製家具這樣複雜多變的行業，也能透過開源、無程式碼工具實現高效、優雅的管理。數位轉型不應只保留給大公司——它應該觸及每一個認真追求成長的小型企業。

![數位轉型](https://static-docs.nocobase.com/7-xckvv2.png)

NocoBase 賦能那些曾經只能旁觀的傳統企業，讓他們終於能夠按照自己的方式建立自己的數位管理系統。我們的使命：讓數位化真正可及、真正可用、真正對每個準備擁抱變革的企業產生影響。每一個用心經營的公司，都值得體驗數位化帶來的成長。
