## 元の投稿

NocoBase チームとコミュニティのみなさんへ

このプラットフォームがどのように私たち家族のビジネスを変えたのか、私の経験を共有したいと思います。

私はメキシコ北部の出身で、もう十年以上、肥料販売やトウモロコシ・豆・ソルガムの栽培を行う自社の業務を一元管理できる仕組みを作りたいと考えてきました。

購買、販売、在庫、物流、プロジェクト、財務、人事、設備保守、燃料管理など、すべてをひとつにまとめられる ERP がずっと理想でした。

しかし、その夢は途中で悪夢に変わりました。

開発者に依頼したものの、必要な機能の三分の一ほどしか満たさないシステムで見積もりは20万ペソ\*。

着手金として 60% を支払いましたが、結局最後まで完成しませんでした。

渡されたソースコードは、技術知識のない私には扱えないもので、お金も戻ってきませんでした。

家族から託されたこのプロジェクトで、自分は失敗してしまったと感じました。

それでも諦めませんでした。

各部署から手作業でデータを集め、Excel に入力して管理を続けましたが、非効率で情報はすぐに散らばり、表は複雑になりすぎて分析も連携もできませんでした。

Access でデータベースを作ろうとしましたが、それもうまくいきませんでした。

理想は現場で使えるモバイルアプリを作ることでしたが、見積もりは VAT 別で 36 万ペソ\*。

一度失敗し信頼も失っていた私には、とても選べる選択肢ではありませんでした。

私はプログラミングの経験がなく、Visual Basic、マクロ、Power Apps、Power BI などもまったく扱えませんでした。努力して ER 図を理解することはできましたが、それが限界でした。

AI ツールが登場してからは、Visual Studio Code を使って学ぼうともしましたが、それでも習得までの道のりはとても険しいものでした。

### 終わりの見えない探し続ける日々

Airtable や NocoDB など、十以上のノーコードツールを試しました。

しかし毎回同じ壁にぶつかりました。ユーザー単位の料金が高すぎる（最低でも10ユーザーは必要）、レコード数や自動化に上限がある、そして何より機能が複雑な業務にはまったく足りません。

諦めかけていました。

そんな中で突然出会ったのが NocoBase でした。

最初は「またよくあるツールだろう」と思っていました。

ですがデモを触った瞬間、それがまったくの別物だと分かりました。本当に根本から違っていました。

NocoBase が本当の解決策になった理由

* 完全無料：コミュニティ版だけで十分強力で、1ペソもかかりません。
* 自己ホスティング：自前サーバーにインストールでき、すべて自分で管理できます。
* 制限なし：ユーザー数、レコード数、自動化の回数などに一切の上限がありません。
* 高い自由度：ドラッグ＆ドロップで必要な画面を思い通りに作れ、コードも不要。
* アクセスが簡単：独自ドメインにつなげ、インターネット接続があればどの端末からでも利用可能。ローカルネットワークならオフラインでも使えます。

### どのように運用を軌道に乗せたのか

私は NocoBase を動かすためだけに、小型の PC を1台購入してサーバーとしてセットアップしました。

* 第8世代 Intel
* i3 / RAM 16GB
* SSD 256GB

価格はわずか 2,350 ペソでした\*。

消費電力も少なく、半年以上、24時間365日動き続けています。

NocoBase を知ったのは 2025年3月。当時はバージョン 1.6 で、ちょうど変数機能やレスポンシブレイアウトが追加され始めた頃でした。

私が考える、NocoBase で強力なアプリを作るための4つの柱は以下です。

1. 整ったデータベース構造
2. 分かりやすい画面設計
3. 無駄のないワークフロー
4. 明確な権限設定

### 夢が現実に

NocoBase のおかげで、ようやく長年の夢を形にすることができました。

思い描いていたすべての機能をひとつにまとめ、しかも変更のたびに開発者に依頼する必要もありません。

今では、ユーザーごとに適切な権限を設定でき、必要な情報だけが見えるようになりました。

その結果、現場からのデータをほぼ誤りなくリアルタイムで取得できるようになりました。

そして何より大きな変化は、集まった情報を分析してより良い意思決定ができるようになったことです。

* **購買最適化：** 仕入先や利益率の高い商品がすぐに分かる。
* **販売・顧客分析：** 価値の高い顧客や離脱した顧客を把握し、データに基づいた提案ができる。
* **在庫管理：** 欠かせない物資が切れないように把握できる。
* **トレンド把握：** 売上の増減が見え、先の計画が立てやすくなる。
* **人員管理：** 従業員一人ひとりの生産性や収益性を測定できる。
* **予防保全：** 設備や機械のメンテナンスを先回りして管理できる。
* **完全なトレーサビリティ：** 各畑・区画ごとの記録を詳細に残せる。

NocoBase チームと、継続的に進化させてくれているコミュニティには本当に感謝しています。

今後はまず Pro 版に、その後 Enterprise 版にアップグレードし、さらに活用の幅を広げたいと考えています。

### これからについて

正式な 2.0 リリースが本当に楽しみです。

Code Agent やデータベースコンテキストに直接アクセスできる機能によって、コードを書かずにアプリをもっとカスタマイズしたり、高度な分析を加えたりできると感じています。

私にとっては、まさに理想的なツールです。

学ぶには時間がかかりましたが、ドキュメントに必要な情報はすべて揃っています。今は本番環境のデータベースをコピーし、2.0 アルファ版のドキュメントを読みながら勉強しているところです。

### 行動への呼びかけ

Google や YouTube、Facebook で「NocoBase」と検索しても、スペイン語の情報がほとんど見つからないことにずっともどかしさを感じていました。

その一方で、機能は少ないのに有料ツールばかりがあちこちで宣伝されています。

そこで私は、NocoBase をスペイン語で学べる YouTube チャンネルを立ち上げることにしました。私のように高額な費用をかけずに ERP を作りたい人の力になれるように。

改めて感謝を伝えたいです。私にとって NocoBase は単なるソフトではなく、十年間の努力がようやく実を結んだ象徴です。

チームのみなさん、そしてコミュニティのみなさんに大きな感謝を。みなさんのおかげで、私は夢を実現することができました。

NocoBase はビジネスアプリケーションの未来だと本気で思っています。その可能性は非常に大きく、必ずトップに立つと確信しています。

特に尊敬しているのは、チームが謙虚で、過剰な宣伝や大げさな約束をせず、人々が自分で価値を見つけられるようにしていることです。他のツールのように、何でもできると謳ってお金だけを求める態度とは全く違います。
