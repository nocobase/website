### 🚀 Verbesserungen

- **[Flow-Engine]** Unterstützt das Auflösen von Werten von Feldern im aktuellen Formulardatensatz, auch wenn diese nicht zum Bearbeitungsformular hinzugefügt wurden. ([#8436](https://github.com/nocobase/nocobase/pull/8436)) von @gchust

### 🐛 Fehlerbehebungen

- **[Flow-Engine]** Behebt, dass Schritte eines dynamischen Ereignis-Flows zweimal ausgeführt werden, wenn ein Popup per Klick auf eine Schaltfläche geöffnet wird. ([#8435](https://github.com/nocobase/nocobase/pull/8435)) von @gchust

- **[Vorlagendruck]** Zeigt Leerzeichenfelder in Version 2.0 an von @jiannx
