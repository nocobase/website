## **No.15: Budibase**

![Budibase.PNG](https://static-docs.nocobase.com/a75c7ed50ec8b1fa840c54aa560da590.PNG)

**Resumen del crecimiento de estrellas:**

* Crecimiento: +725
* Estrellas al 31 de diciembre de 2024: 23,277
* Estrellas al 31 de marzo de 2025: 24,002

**Enlaces:**

* GitHub: [https://github.com/Budibase/budibase](https://github.com/Budibase/budibase)
* Sitio web: [https://budibase.com](https://budibase.com)

**Acerca del proyecto:**

Budibase es una plataforma creada para desarrollar herramientas internas con facilidad. Es especialmente adecuada para casos de uso como el manejo de formularios, flujos de trabajo de aprobación y paneles de datos:

* **Fuentes de datos flexibles:** Conéctate a bases de datos externas como PostgreSQL o MySQL, usa APIs REST o gestiona datos con la base de datos integrada de Budibase.
* **Constructor de UI visual:** El editor de arrastrar y soltar facilita el diseño de diseños responsivos sin escribir código.
* **Automatización integrada:** Define disparadores y acciones para automatizar flujos de trabajo empresariales con un esfuerzo mínimo.

**Ideal para:**

Crear sistemas internos como paneles de administración, flujos de aprobación, aplicaciones de recopilación de formularios o portales para clientes. Budibase acelera el desarrollo y es ideal para adaptarse a las necesidades empresariales en evolución.

**Aspectos destacados recientes:**

En noviembre de 2024, se lanzó la versión 3.0, que incluye varias actualizaciones importantes:

* **Funciones de IA:** Se agregaron herramientas como paneles de configuración de IA, Budibase AI, un generador de CRON impulsado por IA y campos inteligentes para mejorar la automatización y la inteligencia.
* **Ramificación de flujos de trabajo:** Se introdujo soporte para ramificaciones dentro de los flujos de automatización, lo que permite diseños de procesos más complejos y dinámicos.
* **Gestión visual de roles:** Se mejoró la interfaz de RBAC para que la configuración de permisos sea más intuitiva y fácil de usar.

## **No.13: NocoBase**

![NocoBase.png](https://static-docs.nocobase.com/aeea6111d442a314eb7b709d842bfd20.png)

**Resumen del crecimiento de estrellas:**

* Aumento de estrellas: +844
* 31 de diciembre de 2024: 12,533 estrellas
* 31 de marzo de 2025: 13,377 estrellas

**Enlaces:**

* GitHub: [https://github.com/nocobase/nocobase](https://github.com/nocobase/nocobase)
* Sitio web: [https://www.nocobase.com/](https://www.nocobase.com/)

**Acerca del proyecto:**

**NocoBase** es una plataforma de desarrollo no-code/low-code de IA de código abierto con la extensibilidad como núcleo, diseñada para construir y ampliar rápidamente aplicaciones empresariales a través de una arquitectura basada en plugins. En comparación con los enfoques centrados en hojas de cálculo o UI, NocoBase adopta una arquitectura completamente impulsada por modelos de datos, lo que la hace más adecuada para aplicaciones empresariales con flujos de trabajo complejos, permisos detallados y roles claramente definidos.

Lo que hace único a NocoBase:

* **Arquitectura de plugins modular:** Todas las funciones, desde permisos y flujos de trabajo hasta APIs, se cargan como plugins independientes, lo que te permite habilitar, reemplazar o ampliar la funcionalidad según sea necesario.
* **Desarrollo impulsado por modelos:** Cada estructura de tabla y relación de campo se configura mediante la configuración, lo que admite relaciones muchos a muchos, campos anidados y vistas de datos para lógica empresarial compleja.
* **Separación frontend-backend:** El backend se centra en el modelado de datos, mientras que el frontend utiliza bloques de construcción visuales para ensamblar UIs personalizadas, desacoplando la estructura de datos del diseño de la UI.
* **Permisos avanzados:** Ofrece control de acceso basado en roles, condiciones y a nivel de campo, adecuado para sistemas con múltiples roles.
* **Empleados de IA integrados en el sistema:** Integra perfectamente las capacidades de IA en las interfaces de usuario, los flujos de trabajo empresariales y los contextos de datos, lo que permite que la IA se aplique de manera práctica en escenarios empresariales reales.

**Ideal para:**

NocoBase es ideal para desarrolladores y equipos que trabajan con frameworks backend, herramientas internas o sistemas que requieren un modelado de datos enriquecido, flujos de trabajo dinámicos y un control de acceso robusto. Piénsalo como una versión low-code de Strapi, Hasura y un motor RBAC, completamente de código abierto, autogestionable y personalizable con plugins.

**Aspectos destacados recientes:**

El primer trimestre de 2025 trajo varios hitos importantes:

* **Actualización de precios (abril):** Para reducir la barrera de entrada, NocoBase abrió el código fuente de sus plugins comerciales y redujo todos los precios en un 50%.
* **v1.6.0 (marzo):** Se introdujo el modo de implementación en clúster, se mejoraron los protocolos de seguridad y se agregó un administrador de migraciones, mejorando la estabilidad y la escalabilidad.
* **v1.5.0 (febrero):** Se optimizó el rendimiento del núcleo, se agregó soporte para la importación/exportación de datos a gran escala, gestión de correo electrónico integrada y funciones de plantillas de impresión, mejorando el rendimiento y la usabilidad.
